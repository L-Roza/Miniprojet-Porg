# La classe Capteurs
![classe Capteurs](/images/classe-capteurs.png)

1. **min_value** Valeur minimale que le capteur peut mesurer
2. **max_value** valeur maximale que le capteur peut mesurer
3. la fonction **read_value** renvoie un dictionnaire qui contient toutes les informations de la mesure  
   Les mesures sont générées de façon aléatoire
